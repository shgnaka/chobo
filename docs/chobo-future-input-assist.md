# 検索と入力補助

## 1. 目的

入力効率と再発見性を高めるための補助機能を整理する。

## 2. 論点

### 2.1 多条件検索

- 検索軸を固定する
- 組み合わせ条件を定義する
- 大量データでも破綻しない実装にする

### 2.2 テンプレート

- テンプレートの保存単位を決める
- 編集後も再利用できるようにする
- 個人入力に最適化する

### 2.3 候補と自動補完

- 直近の選択値を再利用する
- 候補順のルールを決める
- 誤入力を増やさないようにする

## 3. issue 分割の目安

- `Search`
- `Templates`
- `Autocomplete and suggestions`
