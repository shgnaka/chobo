# 資産管理の拡張

## 1. 目的

今は扱わない資産運用や返済を、別機能として後から拡張できるようにする。

## 2. 論点

- 残高推移だけでなく、引落予定や月末見込みを出す余地を残す
- 多端末同期、銀行連携、OCR、位置起点通知は将来論点として切り分ける
- 投資、ローン、分割払いの厳密化は別 issue として扱う

## 3. issue 分割の目安

- `Future account management`
- `Bank/card/receipt integrations`
- `Loans, installments, and investments`
